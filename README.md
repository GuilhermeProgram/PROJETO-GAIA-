## Sobre o projeto

A **Gaia** é uma plataforma voltada para a economia circular: coletamos resíduos e itens recicláveis e os transformamos em novos produtos com valor real. O nome vem da deusa grega da Terra — porque acreditamos que cuidar do planeta começa por fechar o ciclo dos materiais que já existem.

---

## Como funciona

| # | Etapa | Descrição |
|---|---|---|
| 01 | **Coleta** | Recebemos resíduos e materiais recicláveis de pessoas, empresas e parceiros. |
| 02 | **Triagem** | Classificamos e separamos os materiais por tipo e qualidade. |
| 03 | **Transformação** | Processamos os insumos e os convertemos em novos itens úteis. |
| 04 | **Distribuição** | Os produtos gerados voltam para o mercado, fechando o ciclo. |

---

## Por que a Gaia?

Resíduos não são lixo — são matéria-prima esperando uma nova função. A Gaia existe para provar que é possível gerar valor econômico e ambiental ao mesmo tempo, reduzindo o desperdício e o impacto nos aterros.

## ⚙️ Como rodar localmente

```bash
# clone o repositório
git clone https://github.com/seu-usuario/gaia.git

# entre na pasta
cd gaia

# crie e ative o ambiente virtual
python -m venv venv
source venv/bin/activate

# instale as dependências
pip install -r requirements.txt

# rode as migrações e inicie o servidor
python manage.py migrate
python manage.py runserver
```

Acesse **http://localhost:8000** no navegador. 🚀
